# 🍽️ Catering Website

A modern and responsive catering service website built using HTML, CSS, Bootstrap, and JavaScript.  
This project showcases a clean UI design for food and catering businesses with sections like menu, gallery, customer feedback, staff details, contact form, and FAQ.
